# hpc-bridge Plugin v1 — M1 Implementation Plan

> **For agentic workers:** TDD per task (red → green → commit). Builds on M0 (branch `claude-effort`).

**Goal:** A `run_shell` MCP tool that dispatches a real shell command to the warm endpoint via Globus Compute `ShellFunction` and returns a structured `ShellOutcome`. All Globus interaction behind an injectable `Runner` so the dispatch logic is unit-tested without Globus; one live dogfood proves it end-to-end.

**Architecture:** `dispatch.execute(command, runner)` builds a `ShellOutcome` from a `Runner` result (pure, mockable). `GlobusRunner` holds a long-lived `globus_compute_sdk.Executor` and submits `ShellFunction`s (the real side-effect, in the server lifespan). `run_shell` tool ensures the endpoint is warm, then dispatches; cold → honest `cold_start` outcome.

**Spec:** [`../design/plugin-v1-design.md`](../design/plugin-v1-design.md) §4, §7.

**Files:** Create `src/hpc_bridge/dispatch.py`; Test `tests/test_dispatch.py`

- [ ] Test: a fake runner returning `(returncode=0, stdout="hi", stderr="")` → `execute` returns `ShellOutcome(phase="complete", exit_code=0, stdout="hi", block_state="warm")`. Nonzero rc preserved.
- [ ] Impl:
```python
from __future__ import annotations
from typing import Protocol
from .models import ShellOutcome


class ShellLike(Protocol):
    returncode: int
    stdout: str
    stderr: str


class Runner(Protocol):
    async def run(self, command: str) -> ShellLike: ...


async def execute(command: str, runner: Runner, *, block_state: str = "warm") -> ShellOutcome:
    res = await runner.run(command)
    return ShellOutcome(
        phase="complete",
        exit_code=res.returncode,
        stdout=res.stdout,
        stderr_snippet=res.stderr,
        block_state=block_state,
    )
```

### Task 2: `GlobusRunner` (real dispatch boundary)

**Files:** Create `src/hpc_bridge/runner.py`; Test `tests/test_runner.py`

- [ ] Test: with an injected `executor_factory` returning a fake executor whose `submit(...).result(t)` yields a stub ShellResult, `GlobusRunner(endpoint_id, executor_factory=...).run("echo hi")` returns that stub; the executor is created once (cached) and reused; `close()` calls `shutdown()`.
- [ ] Impl:
```python
from __future__ import annotations
import asyncio


class GlobusRunner:
    def __init__(self, endpoint_id: str, executor_factory=None, timeout: float = 200.0) -> None:
        self.endpoint_id = endpoint_id
        self.timeout = timeout
        self._ex = None
        self._factory = executor_factory or self._default_factory

    def _default_factory(self):
        from globus_compute_sdk import Executor
        return Executor(endpoint_id=self.endpoint_id)

    def executor(self):
        if self._ex is None:
            self._ex = self._factory()
        return self._ex

    async def run(self, command: str):
        from globus_compute_sdk import ShellFunction
        fut = self.executor().submit(ShellFunction(command))
        return await asyncio.to_thread(fut.result, self.timeout)

    def close(self) -> None:
        if self._ex is not None:
            self._ex.shutdown()
            self._ex = None
```
- [ ] Commit.

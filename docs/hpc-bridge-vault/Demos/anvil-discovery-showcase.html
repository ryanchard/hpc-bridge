policy decision

Discovery before provisioning. The model refuses to start a node blind. It gathers live partitions, associations, and QOS caps, then surfaces them as a selection gate — the human picks the partition. The probe also caught a trap: sinfo reports every walltime as infinite, so the model cross-checked the QOS table for the true caps before presenting them.
